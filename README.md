# Cuestionario VixiOn

Cuestionario del Perfil Visual del **Instituto de Ojos de Puerto Rico**.

15 preguntas en 4 secciones que identifican el perfil visual del paciente (Vital, Intacta, Infinita, Órbita, Nova) con flags clínicos para ojo seco, historia refractiva y perfil perfeccionista.

## Demo en vivo

Una vez desplegado: **https://salimperez974.github.io/cuestionario.vixion/**

Modo clínico (muestra respuestas y puntajes): añade `?clinico=true` al URL.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run preview
```

El despliegue a GitHub Pages es automático vía GitHub Actions cuando se empuja a `main`.
